## Minimum-privilege rule

If you lack GitHub permissions/settings required to proceed:

- **Do not workaround.**
- Request the minimum necessary permission explicitly.

## Permission request template

Copy/paste and fill in:

**Request:** <what permission or setting is needed>
**Scope:** <repo/org/environment>
**Reason:** <why it is required for the approved plan>
**Minimum privilege justification:** <why anything less won't work>
**Duration:** <temporary/permanent>
**Auditability:** <how changes remain PR-based / logged>
**Fallback:** <what happens if not granted>
